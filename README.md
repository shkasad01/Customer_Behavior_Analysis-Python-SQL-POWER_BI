📊 Data Analytics Project: Customer Behavior Analysis
🔎 Overview

This project focuses on analyzing customer shopping behavior using a structured data analytics workflow. It covers data loading, exploratory data analysis (EDA), data cleaning, SQL-based querying, and dashboard creation to generate actionable insights.

The goal is to demonstrate end-to-end data analysis skills using industry-standard tools.

📁 Dataset
Name: Customer Shopping Behavior Dataset
Format: CSV
Description: Contains customer demographics, purchase history, and transaction details.
Key Features:
Customer ID
Age / Gender
Product Category
Purchase Amount
Purchase Date
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
PostgreSQL (for structured querying)
Power BI (for dashboard visualization)
Jupyter Notebook
⚙️ Project Steps
1. Data Loading
Imported dataset using Pandas
Checked structure, data types, and basic info
2. Exploratory Data Analysis (EDA)
Summary statistics
Distribution analysis
Correlation checks
Visualization of trends (sales, categories, demographics)
3. Data Cleaning
Handled missing values
Removed duplicates
Corrected data types
Standardized column formats
4. SQL Analysis (PostgreSQL)
Loaded cleaned data into PostgreSQL
Performed queries such as:
Total sales by category
Customer segmentation
Monthly sales trends
Top-performing products
5. Dashboard Creation (Power BI)
Built interactive dashboard with:
Sales overview
Customer insights
Category performance
Time-based trends
📊 Dashboard Highlights
KPI cards (Total Sales, Customers, Orders)
Category-wise sales breakdown
Customer demographics insights
Time-series sales trends
Interactive filters for dynamic analysis
📈 Results & Insights
Identified top-selling product categories
Observed customer purchasing patterns
Detected seasonal trends in sales
Segmented customers based on behavior
